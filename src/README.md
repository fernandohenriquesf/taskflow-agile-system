# TaskFlow Agile System

Sistema web simples para gerenciamento de tarefas utilizando metodologia ágil.

## Funcionalidades

- Criar tarefas
- Excluir tarefas
- Visualizar tarefas

## Tecnologias

- Python
- Flask
- Pytest
- GitHub Actions

## Metodologia

Foi utilizada metodologia Kanban.

## Mudança de Escopo

Inicialmente o sistema teria apenas criação de tarefas.

Após revisão do escopo foi adicionada funcionalidade de exclusão de tarefas.
Tecnologia utilizada: Flask Framework.
